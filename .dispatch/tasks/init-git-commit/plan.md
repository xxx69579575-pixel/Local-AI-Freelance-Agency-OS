# Phase 0.3 — Git 初始化、設定與 First Commit

- [ ] 執行 `git status` 確認 git 已初始化
- [ ] 設定 `git config user.name="dev"` 和 `git config user.email="dev@example.com"`（若尚未設定）
- [ ] 執行 `git config --list` 確認設定已生效
- [ ] 若有未追蹤或未提交的檔案，執行 `git add -A` 並 commit（訊息："chore: first commit — initialize project environment"）；若已有相同 commit 則跳過
- [ ] 執行 `git log --oneline -3` 回報最近三筆 commit
- [ ] 將結果寫入 `.dispatch/tasks/init-git-commit/output.md`
