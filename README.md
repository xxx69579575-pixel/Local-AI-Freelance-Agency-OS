# Local AI Freelance Agency OS

AI-powered freelance agency operating system built and managed entirely through Claude Code.

## Live Demo

https://local-ai-agency-os.vercel.app

## Overview

This project implements an AI-driven freelance agency operating system using:
- TypeScript + Node.js ESM
- Anthropic Claude SDK
- SDD (Spec-Driven Development) methodology

## Development

```bash
npm install
npm run dev        # run with tsx
npm run build      # compile TypeScript
npm test           # run Jest tests
```
